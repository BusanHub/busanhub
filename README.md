# BusanHub

## 1. 개요
- 부산의 모든 정보와 커뮤니티 서비스를 접할 수 있는 프론트엔드 프로젝트

<br>

## 2. 기술 스택
- 프론트엔드 : Vue.js 3 (Vite)
- 배포 : [Netlify](https://busanhub.netlify.app/)

<br>

## 3. 요구사항
- 커뮤니티 기능 (localStorage)
  - CRUD
  - 검색, 조회수, 북마크, 좋아요 (추가)
- 챗봇 기능 (OpenAI API)

<br>
